# Jua Kali Konnect

## Overview
**Jua Kali Konnect** is a web-based platform designed to connect individuals with skilled artisans who can help solve everyday problems. Whether it's plumbing, electrical work, carpentry, or other services, the platform makes it easy to find reliable professionals.

## Live Demo
A full walkthrough: client sign up & login, an artisan (fundi) signing up and setting up their professional profile, browsing and hiring a fundi, the M-Pesa escrow payment flow, and a simulated client satisfaction message.

## Purpose
The main goal of this project is to:
- Help users quickly find skilled artisans
- Provide a convenient way to request services
- Support local talent by increasing visibility for artisans

## Features
- Search for artisans based on skill or service
- User-friendly interface for browsing and selecting services
- Platform designed for accessibility and ease of use

## Installation
This project does not require installation.  
It runs entirely on a web platform and can be accessed through a browser.

## Contribution
Contributions are welcome to improve the platform. One key area for contribution is:

- Adding a **payment system** to allow users to securely pay artisans through the website

### How to Contribute
1. Fork the repository
2. Create a new branch
3. Make your changes (e.g., implement a payment integration)
4. Commit and push your changes
5. Submit a pull request
